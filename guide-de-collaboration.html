Mots de passe

Stocker vos mots de passe n'est pas nécessairement mon dada mais il arrive parfois que cela s'avère nécessaire. Fort heureusement, nous pouvons trouver une solution simple même si cela représente un petit coût pour vous.

Cette solution passe par le logiciel 1password. Ce logiciel est compatible avec Windows/OSX/Android/iOS et permet tout simplement de stocker ses mots de passe automatiquement (ou presque) puis d'y accéder via une application et/ou votre navigateur web.

Le logiciel créer dans un premier temps un mot de passe unique et enregistre les suivants. Il permet également de générer des mots de passe différents pour chaque site et ainsi de sécuriser tous vos accès.

Le logiciel permet également de partager un trousseau d'accès (ensemble de mot de passe) entre différentes personnes via iCloud, Dropbox ou dans un dossier qui pourra être celui associé à Google Drive.

Gestion du code

La question de la gestion et du suivi du code est un vrai problème. J'utilise au quotidien la plateform Github. Vous pouvez vous créer un compte et une organisation vous permettant de bénéficier de repositories privé et donc reservé à certaines personnes (dont moi). Bien entendu, si vous avez votre propre serveur Git, ce point n'est pas un problème.

Le tarif minimum pour cette étape est de 25$/mois et est vivement recommandée ne pouvant pas archiver le code de tous mes projets ad vitam.

Communication et échanges cryptés (email et discussion)

Si vous le souhaitez, nous pouvons protéger nos échanges via un certificat basé sur GPG. Dans un premier temps, il vous faut installer GPGTools pour OSX ou GPG4Win pour Windows.

En installant ce programme, vous pourrez créer votre certificat et ainsi protéger vos accès.

Liste des tutoriels :

  • Mac : It's time to encrypt your Email
  • Chrome/Firefox (si vous utilisez un Gmail/Yahoo! Mail/Outlook.com/...) : Mailvelope
  • Windows (à venir)

A noter qu'un annuaire de clés baptisé keybase est également disponible et vous permet de favoriser vos échanges. Un tutoriel est disponible ici et vous pouvez consulter mon profil via ce lien.

Etant donné que cette opération peut-être technique, n'hésitez pas à me solliciter pour mettre en place ces solutions ou à vous rendre dans un Café Vie Privée.

Concernant les échanges de type chat/appel, je vous conseille d'installer l'excellente suite de logiciels Telegram compatible avec tous les systèmes d'exploitation et smartphones. Il peut être associé à Signal/RedPhone permettant également de chiffrer des communications vocales.
